# neuromodeling-genesis-example
An example of simple visual cortex model in GENESIS 2.4

# Prerequisites
Install GENESIS 2.4 from: https://github.com/genesis-sim/genesis-2.4

# Run simulation
nxgensis visual-net.g

# Plot results
gnuplot plot -p
